📋 Example: User Filters Trades by Symbol "AAPL"

📤 Request Flow

1
User types "AAPL" in filter
handleFilterChange('symbol', 'AAPL')
2
React calls API client
const results = await api.trades.search({ symbol: 'AAPL' })
3
API client makes HTTP request
GET http://localhost:3001/api/trades/search?symbol=AAPL
4
Backend route processes request
const trades = await prisma.trade.findMany({ where: { symbol: { contains: 'AAPL' } } })

📥 Response Flow

5
Database returns filtered trades
[{ id: 1, symbol: "AAPL", price: 150.50, ... }, ...]
6
Backend sends HTTP response
res.json(trades)
7
API client receives data
return response.data
8
React updates UI
setFilteredTrades(results)
