📖 How to Use

1. Upload the JSON file from Program 1 (CSV Statement Analyzer)
2. Click "Start FIFO Matching" to process buy/sell pairs using First-In-First-Out logic
3. Review the matched trades, open positions, and detailed matching log
4. Download JSON (for Program 3) or CSV (for troubleshooting in Excel)

🎯 FIFO Matching Logic

  • • First buy matches with first sell (chronologically)
  • • Handles partial fills and multiple trades per symbol
  • • Creates LONG positions (BUY → SELL)
  • • Creates SHORT positions (SELL → BUY)
  • • Calculates accurate P&L and percentage changes

📊 Output Formats

  • • JSON: Structured data for Program 3 (Duplicate Checker)
  • • CSV: Excel-friendly format for troubleshooting
  • • Log: Detailed matching process for debugging
  • • Statistics: Summary counts and validation
